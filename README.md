* Project Overview

The Food Recipe Finder App is a web-based application that allows users to search for recipes, explore detailed cooking instructions, and discover meals based on ingredients or dietary preferences. This project demonstrates API integration, DOM manipulation, and modern JavaScript practices.

* Objective
  
The goal of this project is to:

Learn how to work with real-world APIs
Practice JavaScript (ES6+) concepts
Dynamically update UI using DOM manipulation
Implement search, filtering, and data handling features
Build a responsive and user-friendly interface

* API Used

This project uses the Spoonacular Food API.
* Documentation: https://spoonacular.com/food-api

* API Capabilities:
  
Search recipes by name or ingredients
Get detailed cooking instructions
Fetch nutritional information
Filter recipes by diet (vegan, keto, etc.)

* Features
  
  1. Recipe Search
Search recipes by keywords (e.g., "pasta", "cake")
Fetch results in real-time using the Fetch API
  2. Search by Ingredients
Enter available ingredients
Get recipes that can be made with them
  3. Recipe Details
Click on a recipe to view:
Ingredients list
Step-by-step instructions
Nutrition details (calories, protein, etc.)
  4. Filters & Categories
Filter by:
Diet (Vegan, Vegetarian, Keto)
Cuisine
Meal type (Breakfast, Lunch, Dinner)
